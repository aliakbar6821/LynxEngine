## ⚠️ Important Notes
 1. 💾 **ALWAYS** backup your original framework.jar before making any modifications.
 2. 🔀 **Custom ROM users:** Make sure to add Lynx hooks **BEFORE** any existing spoofing code.
 3. ⏩ **Port ROM users:** Just add the Lynx hooks as shown (there is no existing code to worry about).
 4. 🔍 **Line numbers may vary:** Search for the specific code patterns shown above rather than relying on line numbers.
 5. 🧪 **Testing:** Verify the installation using Play Store developer options after booting.

## 🆘 Emergency Recovery
If your device refuses to boot (bootloop):
 1. 🔄 Boot into Recovery (TWRP/OrangeFox, etc.).
 2. 🧹 Dirty flash your ROM (If using a port, just flash the specific partition).
 3. 🚀 Reboot.
